CPU Endpoint

Retrieve complete specifications for any CPU by brand and model.

Request

GET https://nucleus.api.qingyi-studio.top/cpu/{brand}/{model}

Parameter Type Description
brand string Manufacturer (e.g., "intel", "amd")
model string CPU model number (e.g., "i5-12500H", "R9-7950x")

Error Handling

Your backend implements these exact error cases:

1. CPU Not Found (404)

Triggered when: os.IsNotExist(err)

HTTP/1.1 404 Not Found
Content-Type: text/plain

CPU information not found

2. File Read Error (500)

Triggered when: os.ReadFile fails

HTTP/1.1 500 Internal Server Error
Content-Type: text/plain

Error reading CPU information

3. JSON Parse Error (500)

Triggered when: json.Unmarshal fails

HTTP/1.1 500 Internal Server Error
Content-Type: text/plain

Error parsing CPU information
